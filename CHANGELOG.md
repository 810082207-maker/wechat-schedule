# 更新日志

## v1.1.0 (2026-03-26)

### ✨ 新增功能

#### 订阅消息提醒功能
- [x] 添加订阅消息服务 (`miniprogram/utils/subscribe.js`)
- [x] 创建发送提醒云函数 (`cloudfunctions/send-reminder/`)
- [x] 创建定时触发器云函数 (`cloudfunctions/reminder-trigger/`)
- [x] 添加配置指南 (`docs/REMINDER_CONFIG.md`)

### 📝 功能说明

#### 用户体验流程
1. 用户创建日程时，可选择开启提醒
2. 点击「开启提醒」按钮，弹出微信订阅授权框
3. 用户点击「允许」后，系统会在指定时间发送微信通知
4. 用户收到微信服务通知，点击可直接进入日程详情

#### 技术实现
- 前端：使用 `wx.requestSubscribeMessage` API 请求授权
- 后端：云函数定时检查并发送订阅消息
- 存储：新增 `reminders` 集合存储提醒任务

### 🔧 需要配置的内容

1. 在微信公众平台获取订阅消息模板ID
2. 替换云函数中的模板ID
3. 部署云函数
4. 创建 `reminders` 数据库集合
5. 配置定时触发器（可选）
