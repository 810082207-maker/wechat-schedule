# 🚀 部署指南

## 📋 前置准备

### 1. 注册微信小程序账号

访问 https://mp.weixin.qq.com，按照步骤注册小程序账号。

**需要准备的材料：**
- 企业营业执照或个人身份证
- 企业法人或个人身份验证
- 小程序名称和简介

### 2. 获取 AppID

1. 登录微信小程序后台
2. 进入「设置」→「基本设置」
3. 复制 AppID

### 3. 下载开发工具

下载微信开发者工具：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

## 🔧 本地开发配置

### 步骤 1：克隆项目

```bash
git clone https://github.com/your-username/wechat-schedule.git
cd wechat-schedule
```

### 步骤 2：打开项目

1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择项目目录（wechat-schedule）
4. 填写 AppID
5. 点击「导入」

### 步骤 3：配置云开发

#### 3.1 开通云开发

1. 在微信开发者工具中，点击顶部菜单栏的「云开发」
2. 点击「开通」按钮
3. 选择「基础版」（免费，足够个人使用）
4. 创建环境

#### 3.2 获取环境 ID

1. 在云开发控制台中，点击「环境」
2. 复制环境 ID（格式如：`prod-xxx`）

#### 3.3 配置环境 ID

编辑 `miniprogram/app.js`，替换环境 ID：

```javascript
wx.cloud.init({
  env: 'your-cloud-env-id', // 替换为你的环境ID
  traceUser: true,
});
```

### 步骤 4：创建数据库集合

1. 在云开发控制台中，点击「数据库」
2. 点击「添加集合」，创建以下集合：
   - `users`
   - `schedules`

3. 为每个集合设置权限规则：

**权限规则（JSON）：**
```json
{
  "read": "auth.openid == doc._openid",
  "write": "auth.openid == doc._openid"
}
```

这样可以确保用户只能访问自己的数据。

### 步骤 5：部署云函数

1. 在微信开发者工具中，右键点击 `cloudfunctions/auth` 文件夹
2. 选择「上传并部署：云端安装依赖」
3. 等待部署完成
4. 重复步骤 1-3，部署 `schedule-crud` 和 `reminder` 函数

### 步骤 6：添加图标

在 `miniprogram/images/` 目录下添加以下图标（建议尺寸 81×81px）：

- `today.png` / `today-active.png` - 今日日程图标
- `calendar.png` / `calendar-active.png` - 日历图标
- `profile.png` / `profile-active.png` - 个人中心图标
- `default-avatar.png` - 默认头像

**图标设计建议：**
- 格式：PNG，透明背景
- 颜色：
  - Active 版本：使用主题色 `#4A90E2`
  - 非 Active 版本：使用灰色 `#999999`

### 步骤 7：编译运行

1. 点击开发者工具中的「编译」按钮
2. 在模拟器中预览小程序
3. 测试各项功能

## 📱 真机测试

### 1. 生成体验版二维码

1. 在微信开发者工具中，点击「预览」
2. 使用微信扫描生成的二维码
3. 在真机上测试小程序

### 2. 常见问题排查

**问题：云函数调用失败**
- 检查环境 ID 是否正确配置
- 确认云函数已成功部署
- 查看云开发控制台的日志

**问题：数据库查询返回空**
- 检查数据库权限规则是否正确
- 确认用户已登录（获取 openid）
- 查看数据库中是否有数据

**问题：图标不显示**
- 确认图标文件已添加到 `miniprogram/images/` 目录
- 检查 `app.json` 中的图标路径是否正确
- 尝试重新编译

2. 小程序即可在微信中搜索到
3. 用户可以下载使用

## 🔐 安全建议

### 1. 数据库安全

- ✅ 已配置权限规则，用户只能访问自己的数据
- ✅ 使用云开发的 `_openid` 自动注入，防止数据泄露

### 2. API 安全

- 云函数自动验证用户身份
- 所有操作都需要用户登录

### 3. 隐私保护

- 不收集用户不必要的信息
- 用户数据存储在云开发数据库中
- 遵守微信小程序隐私政策
